# ⚡️ Sokbo (속보 전용 알림 서비스)

> **"뉴스에 속보가 뜨는 순간, 당신의 폰에 먼저 도착합니다."**

Sokbo는 불필요한 정보, 복잡한 설정, 광고성 뉴스 없이 **오직 네이버 뉴스의 '속보'만 실시간으로 배달**하는 초경량 알림 서비스입니다.

## ✨ 제품 철학
- **단순함**: 설정할 것도, 고를 것도 없습니다. 앱을 설치하는 순간 모든 준비가 끝납니다.
- **즉시성**: 10분마다 네이버 뉴스를 스캔하여 '속보'가 포함된 기사를 감지하면 즉시 푸시를 보냅니다.
- **본질 집중**: 기사 요약, 저장, 댓글 등 군더더기를 덜어내고, 알림을 클릭하면 즉시 뉴스 원문으로 연결합니다.

## 📱 주요 가치
- **필터링 없는 정직함**: '속보'라는 이름이 붙은 기사라면 놓치지 않고 전합니다.
- **정보 다이어트**: 24시간이 지난 정보는 자동으로 삭제하여 앱을 항상 최신 상태로 유지합니다.
- **어디서나**: PWA 기술을 활용해 어떤 플랫폼에서든 가볍게 속보를 받아볼 수 있습니다.
