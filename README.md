# 🚀 Sokbo (초경량 관심사 속보 알림 서비스)

> **"가장 빠르고, 가장 가볍게. 당신의 관심사만 골라 속보를 전합니다."**

Sokbo는 넘쳐나는 뉴스 속에서 사용자가 설정한 관심사(증시, 부동산, 반도체 등)에 대한 **'진짜 속보'**만 골라 즉시 배달하는 초경량 알림 앱입니다.

## ✨ 제품 철학

1. **지연 없는 속보**: 복잡한 처리 과정을 생략하고 네이버 뉴스 API와 직접 연동하여 가장 빠른 타이밍에 알림을 보냅니다.
2. **최소한의 개입**: 기사 상세 페이지, 댓글, 요약 기능 등 사용자의 시간을 뺏는 군더더기를 모두 제거했습니다. 알림을 클릭하면 즉시 뉴스 원문으로 연결됩니다.
3. **직관적인 개인화**: 복잡한 설정 없이 관심 있는 프리셋만 선택하면 즉시 나만의 뉴스 피드가 완성됩니다.

## 📱 주요 가치

- **스마트 필터링**: 뉴스 제목에서 HTML 태그를 제거하고 '속보' 키워드를 판별하여 고순도의 정보만 제공합니다.
- **언제 어디서나**: PWA(Progressive Web App) 기반으로 설계되어 모바일(iOS/Android)과 데스크톱 모두에서 네이티브 앱과 동일한 알림 경험을 제공합니다.
- **중복 제로**: 이미 확인한 소식은 다시 보내지 않아 불필요한 알림 피로도를 최소화합니다.

## 🗺 제품 로드맵

Sokbo는 현재 웹 기반 MVP 단계를 넘어, 더 안정적인 푸시 환경과 고도화된 사용자 경험을 위해 **네이티브 앱 패키징 및 스토어 출시**를 목표로 하고 있습니다.

## 📦 배포 및 무료 10분 주기 설정

1. **Vercel 배포**: GitHub 저장소를 연결하고 환경 변수를 등록합니다. (`CRON_SECRET` 필수)
2. **Supabase 트리거 설정**: Vercel 무료 플랜의 제한을 피하기 위해 Supabase를 스케줄러로 사용합니다.
   - Supabase SQL Editor에서 아래 쿼리를 실행하세요.

```sql
-- 1. HTTP 요청 확장 활성화
CREATE EXTENSION IF NOT EXISTS pg_net;

-- 2. 10분 주기 트리거 등록 (URL과 SECRET을 본인 설정에 맞게 수정)
SELECT cron.schedule(
  'sokbo-10min-trigger',
  '*/10 * * * *',
  $$
  SELECT net.http_get(
    url := 'https://여러분의-앱-이름.vercel.app/api/cron',
    headers := jsonb_build_object('Authorization', 'Bearer 여러분의_CRON_SECRET_값')
  );
  $$
);
```
